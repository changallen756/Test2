首頁本身
作品集的首頁上方是模仿皮克斯動畫公司的首頁做成的，具有RWD的功能，標頭選單會隨著視窗寬度收合或展開
文集
這是一個可以呈現許多文章的網站，並且一樣有提供RWD的功能，內容會隨著視窗寬度重新排列
手機規格
這是一個可以顯示多支手機資料的網站，點選手機圖片可以看到該支手機的規格，並且提供修改、刪除資料的按鈕，首頁上方也有新增、搜尋的功能
Honda clone
在這個模仿Honda首頁的網站裡面，提供了置底的選單，游標如果移到上面，選單會向上提起，另外左側的選單也有類似的功能，游標經過時會向右展開。這是一個有RWD功能的網站，視窗寬度變窄的時候選單和內容都會重新排列
MS clone
這是模仿微軟的首頁做出的網站，主要特色是有RWD的功能，瀏覽器變窄的時候，標頭選單會收合，並且中間部分的圖片和頁尾的選單也會重新排列
acer clone
這個網站則是模仿acer的首頁，功能和前一個網站類似，但是多了頁尾的選單在瀏覽器變窄的時候，會變成可折疊的樣式之功能
